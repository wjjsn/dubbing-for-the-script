# 人物设定信息补充
- 爱蜜莉雅：半精灵，身材纤细，是有着一双紫绀色眼眸的银发美少女，和传说中的莎缇拉外貌一模一样。她身着紫白色相间的服饰，银白色的秀发上别着一朵小白花。
- 碧翠丝：发色是淡奶油色，发型是螺旋双马尾的娇小女孩，蓝色的眼睛中心有蝴蝶的花纹。身着饰有粉色丝带的红色褶边连衣裙，搭配粉紫相间的条纹连裤袜和洋红色鞋子。头部的右后方戴有一顶小皇冠，胸口处则用蝴蝶型的丝带系着她的披风。口癖是呢、啦、呀、实际上。
- 罗兹瓦尔：一个高大、英俊、身材健壮的男人。他留着靛青色的头发，长着一对异色瞳，分别是蓝色的右眼和黄色的左眼。他的眼睑用紫色的眼影装饰着，但是相比右眼，他左眼的眼影要更加长一点，覆盖了整个眼睛周围。罗兹瓦尔的着装与常人有许多差别，譬如一顶黑色的帽子，紫色调的衣服，红黑相见的靴子与斗篷，使得他看起来像是一个小丑。不仅如此，他的右脚踝上还戴有铁链。罗兹瓦尔有时也会不化妆出现，尽管他极少这么做。罗兹瓦尔的脸不化妆时在大多人看起来都非常苍白和骇人，然而他的眼睛流露出来的是温柔和恭迎的神情，这种姿态与他化了妆的时候相比完全不一样。需要展现正经面貌时，他也通常会穿着相当符合其贵族身份的黑色礼袍。
- 拉姆：拉姆有着一头及肩的粉色头发，左眼常常被头发遮住而露出右眼；红色的眼瞳以及美丽的五官；在她的头右侧有一个花型的饰带，头上戴着女仆发箍。除了发色，眼睛颜色和饰带颜色以外和她妹妹雷姆看起来几乎一模一样。
- 雷姆：雷姆有着水蓝色短发，下垂的刘海遮住了右眼睛，左侧的刘海用发卡固定，头戴女仆的发带，左边的头发上还别有花型发带。除了颜色不同，雷姆与她的姐姐拉姆看起来一模一样。
- 奥托：一头蓬松的灰发，发丝顺着脸庞垂落至下巴下方。神情严肃时有着稍显帅气的一面，然而通常情况下他不是在回嘴、大喊大叫，就是一脸慌张，破坏了自己的形象。体格中等，因经商原因一直有在自我锻炼。身穿系有腰包的绿色上衣，披着饰由红色蝴蝶结固定的深绿色连帽斗篷。常戴的绿色帽子造型类似苏联毛帽，侧面装饰着白色羽毛，帽子与衣服边缘皆点缀着金边。
- 弗雷德莉卡：弗雷德莉卡·鲍尔曼是罗兹瓦尔麾下的一名女仆，爱蜜莉雅阵营的一员。弗雷德莉卡有着一头淡金色的长发和绿色的眼睛，身高比昴略高一点。唯一与美丽五官不相符的是张嘴时的一口尖牙，这使得她眯起眼时看起来简直就像准备捕食的肉食动物一样。
- 加菲尔：不高，一头金发向后梳拢，有一双眼神锐利的翡翠色眼睛，由于亚人血统，他有着和弗雷德莉卡一样的锋利牙齿。事实上，加菲尔和弗雷德莉卡的五官十分相似，他们最大的不同就是加菲尔额头前那「X」状的疤痕。在亚人血统和勤加锻炼之下，加菲尔的体格十分健壮。平常穿着一条黑色短裤和围在腰上的外淡紫内红蓝的披风以及黑色的小背心，有时他也会戴着长袖黑手套，在他的脖颈处系有一颗蓝色的辉石。加菲尔十分重视感情，和姐姐弗雷德莉卡不同，加菲尔用了母亲的姓——汀泽尔以此永远记住母亲。他有很强的责任感，刚开始加菲尔十分敌视昴，是因为他认为外面的世界无比危险，他想要圣域的家人们能够平安无事，因此他反对解放圣域来以此保护家人们。然而，在被昴击败并在拉姆和昴的支持下加菲尔通过了第一试炼成功面对了痛苦的过去，由此和昴和解，并称呼昴为「大将」（老大）。加菲尔很喜欢读书，所以会说许多奇奇怪怪的俗语和谚语。
- 佩特拉：有着一头泛红棕色的及肩短发以及孔雀蓝色的双眼。在阿拉姆村居住时她常以便服示人，但是搬进大宅以后便因为工作原因改穿女仆服了。魔兽事件中因为盲目信任梅莉而让昴受伤，昴却温柔地原谅了她。佩特拉明白了自己只是被大家温柔相待，对昴暗生情愫。
- 库珥修：威风凛凛的女士，她深绿色的头发留到背部中央，用白色蝴蝶结在末端绑住。她高挺的鼻梁很漂亮，有着琥珀色的双眸。虽然她有着柔和的面容，但表情比较严肃，给人以精悍的印象。她的身材很好，但平时喜欢男装打扮。当她穿礼服温柔地微笑时，她给人的印象大为不同。年幼时，库珥修将深绿长发用粉红色蝴蝶结扎成高马尾。她穿着高级的草绿色礼服，戴着珍珠项链。因为弗利艾的话将头发放下。后来在菲莉丝的支持下，库珥修开始穿男装。库珥修常穿的正装包括带有白色滚边和金色肩饰的蓝黑色长外套，外套底下的白色花边衬衫，蓝黑色长裤，带有白绑带的黑色及膝高跟筒靴，白手套以及中间有一白色条纹的蓝色帽子。她长外套的外翻袖口为酒红色，长外套内侧及在她左臂上的臂带都是同色调的红色。臂带上有象征卡尔斯腾家族的狮子图案。饰物包括七个金色十字花扣——两个各在两边袖口上，四个是在她腹部以金色饰绳连接的双排扣，一个在胸花上以饰绳连到肩膀。在作战时，她穿上镶金边的铁铠，包括胸甲、肩甲和臂甲。她的头发被束成马尾，戴上棕色手套[2]，身披亮红色的披风。
- 菲莉丝：菲莉丝有着亚麻色短发和黄色瞳孔，日常装扮是蓝白色蝴蝶结发饰、挂有蓝白色丝带的蓝色项圈、蓝白色连衣裙、黑色紧身裤、蓝白色长袜、蓝色鞋子、蓝白色护臂，右臂上系着一条蓝色围巾。菲莉丝的猫耳、猫尾源自祖先的亚人血统。无论穿什么服装，菲莉丝总是戴着一个由蓝白色缎带做成的蝴蝶结发饰，这条缎带是库珥修赠予的礼物。
- 威尔海姆：身形高大，有着让人感觉不到暮年的强健身体以及挺直的腰杆。他茂密的白发被仔细地梳拢至后方，白胡子也很整齐，站姿看上去十分优雅。柔和的面容上刻着温柔的皱纹，湛蓝的眼睛有锐利的光芒。身着内里为蓝色的黑色管家服，白色衬衫和白色马甲，还有白手套和黑色领结。
- 普莉希拉：她的言行非常傲慢，与身边的人说话的态度高高在上。然而，她也展示出自己并不愚蠢，而且实际上有跟言行相称的力量。长长的金红色头发流离光彻，宛若太阳，在左边盘起一个小发髻，剩下的长发散落成侧马尾，再以黑色发网、缀以红边的花朵和缎带点装饰发髻，马尾中间以一个小黑色发圈固定。血红双眸隐约闪烁着与她强势性格相符的强大意志。肌肤胜雪，粉唇点缀其上更是锦上添花。旁人公认她美得有有如女神下凡。凡人即使穷一生时间和运气也找不到第二個人像她一样光彩照人。更甚的是，她身材娇小玲珑却拥有引人注目的丰满双乳，婀娜多姿的体态让本来就已超绝的魅力更上一层楼。脚踏五寸高跟鞋，让她和爱蜜莉雅同高。普莉希拉的衣着与她绯红眼眸相得益彰。她常穿的洋装裸露香肩和雪背，红色为主调，黑色蕾丝滚边，缀以小颗翡翠，下摆蓬松长及脚踝，内里一件黑色衬裙，腰部右边饰有大大的红黑相间蝴蝶结，左边则是镶有翡翠的红边黑底玫瑰。裙子下面，是半透明红棕色长筒吊带丝袜，上方的弹力带为黑色。她亦穿着黑色贴颈项链，赤红高跟鞋，以及黑色露指长手套。她随身携带一把绯红折扇，不用的时候放在双峰之间。两根中指上都戴着镶绿玉的指环，从耳垂垂下镶有同样颜色的泪滴形耳坠。她佩戴的首饰皆为做工一流的工艺品，任何人一看就知道价值不菲。
- 阿尔：阿尔身着佛拉基亚产的黑色头盔，绿色马甲，黄褐色裤子拖鞋和橙色斗篷。他戴着头盔以掩盖脸上的伤口。在佛拉基亚的时候因为某事件只剩下一只手。他将剑置于背后的剑鞘内。阿尔性格轻浮愚笨，已经在另一个世界度过了十年时光。他跟菜月·昴可以很轻松地相处。尽管是普莉希拉的骑士，他并不真的将自己当作一个骑士，而是视自己为她的看门人，而普莉希拉自己则把他当作一个杂活工和取悦自己的傻瓜。他对他们的关系亦毫无顾虑。他吹嘘自己唯一拿得出手的本事是让人毫无防备地智取别人。
- 菲鲁特：外表娇小，留着一头中长金发，头上戴着一个黑色领结，笑起来的时候有很明显的虎牙。如红宝石般的红瞳，正如卢克尼卡王族的特征——金发红瞳。她戴着一条深红色的围巾，穿着背心、抹胸以及一条左半边破烂到不复存在的裤子，腰间挂着一个她常用的刀具包。在莱茵哈鲁特带走她后，她被迫穿着一件黄色礼裙和红色的皮质靴子和袖子，扎着橙色的圆发髻，让她看起来够格参加王选。王选开始之后，莱茵哈鲁特给菲鲁特定制了一套新服装，和之前的贫民窟装扮类似，轻量级满分，黑色的上衣以及蓝色的披肩和棕色夹克衫，腰间也挂着一个纳物袋。
- 莱茵哈鲁特：一头宛如炽烈燃烧的火红色头发和倒映着天空般清澈的蓝色眼眸，纤细修长的身材，端正到异常的五官也以那凛然的姿态作为后盾，光是一瞥就知道他是个了不起的人物。被认做是优秀骑士的典范，因为如此出众的外貌、地位以及实力，他受到人们的热烈欢迎，尤其是处于思春期的少女们。他通常身着以白色为主色、红色为辅色、点缀着金色配饰的近卫骑士团的制服，在制服内的则是剪裁合身的黑色服装。但闲暇之时，他常穿着「非正式服装」——依然是制服，但是没有骑士团那件那么精致，也是以白色为主色，不过辅色是紫色，在左肩膀上纹有阿斯特雷亚家的家纹。自成为当代剑圣以来，莱茵哈鲁特就一直将龙剑挂在腰间。
- 安娜塔西亚：一个浅葱眼睛的娇小女子，长长的波浪卷发一直铺到臀部，穿一袭白色长裙，戴毛皮帽子围巾，还有一个皮夹杨的大钱包，头发上箍着星形发夹，脖子上挂着一枚小小的绿色坠饰。在和菜月·昴会谈的时候，戴着一顶粉色小帽，围着蓝色围巾。她被描述为外表可爱，有类似小动物的气味。
- 由里乌斯：身材精瘦高挑的端正美青年，但在衣服下隐藏着发达的肌肉；一头紫发整齐地梳理在后，只有一缕头发垂在脸上；在脸上的是眼尾微微下垂的细长眼睛，而里面有着足以魅惑异性的琥珀色瞳孔；有着兼具高贵和精悍的相貌、优雅的嗓音与谈吐。由里乌斯在大多数出场时都穿着近卫骑士团的正式白色制服，在腰上挂着一把类似西洋剑的细剑；但在和昴决斗时，他脱下了制服而换成了一件有蓝边的黑色西装，使用的是也是木剑。由里乌斯平常总是保持着凛然的姿态与优雅的行为，让人能一眼看出他是一名优秀的骑士；此外，由里乌斯喜欢在思考或者交谈时抚摸自己前端的刘海。
- 里卡多：里卡多是一个两颗大獠牙挂在嘴边的大块头狼人，绿色的眼睛，强健的体型，体毛是棕色短毛，棱角分明的暗棕色鬃毛上装饰着mohican，旁边是两条小型的红色穗带。胸部、二头肌、嘴巴和手掌处是黄褐色毛发，胡子是暗棕色的。
- 蜜蜜：安娜塔西亚的追随者之一，铁之牙佣兵团三位副首领之一。蜜蜜眼睛是蓝色的，橙色长发束成两根麻花辫。她有橙色的猫耳和尾巴，穿着白色长袍。
- 缇碧：缇碧蓝色的眼睛，中等长度的橙色头发。他有橙色的猫耳和尾巴，穿白色长袍。和他哥哥姐姐不同的是，他左眼戴单片眼镜，使他散发出精明智慧的气息。
